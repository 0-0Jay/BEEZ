## 개요 및 개발동기

###  프로젝트 개요
> 일정관리, 업무 통계, 저장소 연동, 위키 등  **IT 프로젝트 환경에서 필요한 기능들을 하나의 플랫폼에서 제공** 
> 프로젝트와 일감을 **체계적으로 관리**하고, 프로젝트 구성원들의 **의사결정과 업무 분배에 도움**을 주는 서비스를 개발

- **Spring** 기반 서버, **Vue.js** 프론트엔드, **OracleDB**를 이용한 실시간 데이터 처리 및 저장, **S3**를 이용한 파일 관리

###  개발 동기
- IT 프로젝트 환경에서 일감 관리, 일정 추적, 의사소통, 버전 관리가 각기 다른 도구에 분산 되어 업무 흐름에 비효율이 발생되는 문제를 해결
- 비효율을 해소하고 프로젝트 전 과정을 하나의 플랫폼에서 통합 관리할 수 있는 시스템의 필요

### 기대 효과
- 프로젝트와 일감의 계층형 구조와 시각적 일정 관리 기능으로 **프로젝트 전체 현황을 한눈에 파악**하고 업무 지연 등의 **문제점을 초기에 발견하여 대응**
- 실시간 소통 기능과 Git 저장소 연동 기능으로 **협업 품질을 높이고**, 보고서 기반의 데이터로 프로젝트 구성원들의 **객관적인 의사결정 지원**
